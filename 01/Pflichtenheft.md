# Pflichten Heft

### 1.Zielbestimmungen
#### 1.1 Musskriterien
- Der Benutzer soll neue Kontakteinträge zu Mitarbeitern und Firmen anlegen können
- Der Benutzer soll Kontaktinformationen im Nachhinein ändern können
- Der Benutzer soll Kontakte Löschen können
- Der Benutzer soll sich Kontakte anzeigen lassen können
- Der Benutzer soll Kontakte nach Eigenschaften(Name, Adresse...) sortieren können

#### 1.2 Wunschkriterien
- Übersichtliche GUI

#### 1.3 Abgrenzungskriterien
- Keine Funktionen zum Nachrichten senden

### 2.Produkteissatz
#### 2.1 Anwendungs Bereiche:
- Diese App kann verwendet werden, um Kontakte zu verwalten und diese in übersichtlicher Form zu Speichern

#### 2.2 Zielgruppe:
- Felix

### 3.Produktumgebung
- Für mobile Geräte (IOS und Android)

### 4.Produktdaten
/D001/ Mitarbeiterdaten: Alle Informationen zu einem Mitarbeiter:
- Name
- Nachname
- Adresse
- Telefonnummer
- Email

/D002/ Firma: Alle Informationen zu einem Mitarbeiter:
- Name
- Adresse
- Mitarbeiter

### 5. ProdukFunktionen
#### 5.1.Speichern
/F001/ Mitarbeiter einfügen:
- Neuer Mitarbeiter wird angelegt. Mittels eines Formulars können gleich die richtigen Daten, wie Name, Adresse etc. angegeben werden.

/F002/ Firma einfügen:
- Neue Firma wird angelegt. Mittels eines Formulars können gleich die richtigen Daten angegeben werden.


#### 5.2.Ändern
/F003/ Mitarbeiterdaten ändern:
- Mitarbeiterdaten werden, wie vom Nutzer spezifiziert geändert.

/F004/ Firmendaten ändern:
- Firmendaten werden, wie vom Nutzer spezifiziert geändert.

#### 5.3.Löschen
/F005/ Mitarbeiterdaten Löschen:
- Der Mitarbeiter wird gelöscht

/F006/ Firma Löschen:
- Die Firma wird gelöscht
- Der Nutzer wird gefragt, ob auch alle Mitarbeiter der Firma gelöscht werden solleb

#### 5.4.Anzeigen
/F007/ Mitarbeiterdaten anzeigen:
- Die Mitarbeiterdaten, die zu einem Mitarbeiter gehören sollen in einer übersichtlichen Art und Weise angezeigt werden können

/F008/ Firma Anzeigen:
- Die Firmendaten, die zu einer Firma gehören sollen in einer übersichtlichen Art und Weise angezeigt werden können

/F009/ Firmen Sortieren:
- Die gespeicherten Firmen sollen nach folgenden variablen sortiert werden:
1. Name

/F010/ Mitarbeiter Sortieren:
- Die gespeicherten Mitarbeiter sollen nach folgenden variablen sortiert werden:
1. Name
2. Firma
3. Adresse

### 7.Benutzeroberfläche
- Startbildschirm: Eine Übersicht, in der man auf ein Mitarbeiter Menü und ein Firmen Menü wechseln kann
- Mitarbeitermenü: Menü, indem man Mitarbeiter hinzufügen kann, sie löschen und verändern kann.
- Firmenmenü: Menü, indem man Firmen hinzufügen, löschen verändern kann.
